# かなうち電鉄 修正版

## 修正内容

- ホーム画面を「つづきから発車」中心のシンプルな画面に変更
- ホーム画面に小さな走行電車の演出を追加
- 運転画面に線路・駅・走る電車・背景の流れを追加
- 入力の進み具合に合わせて電車が駅へ近づくように変更
- 正解時は「定刻到着！」として駅に停車する演出を追加
- ミス時は電車が揺れて「遅延 +1分」を表示
- 時間切れ時は駅を通過する演出を追加
- 運転中のお題表示は「を」「し」「でんしゃ」のように文字だけを大きく表示
- `wo / o どちらでもOK` などの別表記は、運転中に常時表示せず、ヒントまたは正解後に表示
- 簡易路線マップを追加
- スマホ表示向けに情報量を削減

## 起動方法

`index.html` をブラウザで開いてください。

## 音声素材について

`sounds/` 内の音声はテスト用です。実運用では、かなうち電鉄オリジナル素材へ差し替えてください。

## データ管理

問題データとアナウンス文は `data/` 以下に分離しています。

- `data/kana-data.js`
- `data/word-data.js`
- `data/sentence-data.js`
- `data/line-data.js`
- `data/announcements.js`
