# かなうち電鉄 修正版

## 修正内容

- ホーム画面を「つづきから発車」中心のシンプルな画面に変更
- ホーム画面に小さな走行電車の演出を追加
- 運転画面に線路・駅・走る電車・背景の流れを追加
- 入力の進み具合に合わせて電車が駅へ近づくように変更
- 正解時は「定刻到着！」として駅に停車する演出を追加
- ミス時は電車が揺れて「遅延 +1分」を表示
- 時間切れ時は駅を通過する演出を追加
- かなうち本線は練習用として時間制限なしに変更
- 運転中のお題表示は「を」「し」「でんしゃ」のように文字だけを大きく表示
- `wo / o どちらでもOK` などの別表記・入力ヒントは、出題開始時から表示
- 簡易路線マップを追加
- スマホ表示向けに情報量を削減
- 音声ファイルの音量を上げ、ブラウザの音声再生ブロック時に代替音を鳴らす処理を追加

## 起動方法

ZIPを展開してから、フォルダ内の `index.html` をブラウザで開いてください。ZIPの中を直接開くと、ブラウザによっては音声ファイルが読めない場合があります。

## 音声素材について

`sounds/` 内の音声はテスト用です。前版より音量を上げています。音が出ない場合は、設定画面の「テスト再生」を押し、端末の音量・ミュート解除も確認してください。実運用では、かなうち電鉄オリジナル素材へ差し替えてください。

## データ管理

問題データとアナウンス文は `data/` 以下に分離しています。

- `data/kana-data.js`
- `data/word-data.js`
- `data/sentence-data.js`
- `data/line-data.js`
- `data/announcements.js`
