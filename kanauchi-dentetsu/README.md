# かなうち電鉄

ひらがな・単語・文章をローマ字で入力し、電車を定刻運行させるタイピング学習ゲームです。

## 起動方法

`index.html` をブラウザで開いてください。外部ライブラリは不要です。

## 実装済み

- かなうち本線
- ことばひら線
- 文字港レトロ線
- 一文線
- 車内アナウンス線
- うちかいどう新幹線
- やりなおし回送線
- レジェンド線
- Lv.100 / 称号 / EXP
- 遅延ゲージ / 連続定刻 / 運行評価
- バッジ
- ローマ字案内 / 逆引き検索
- localStorage保存
- 音声ファイル再生
- 音設定
- アナウンス文の別ファイル管理

## ファイル構成

```text
kanauchi-dentetsu/
  index.html
  style.css
  script.js
  data/
    kana-data.js
    word-data.js
    sentence-data.js
    line-data.js
    announcements.js
  sounds/
    *.mp3
```

## 音声について

同梱の `sounds/` は、権利上安全なテスト用の簡易効果音です。実運用では、かなうち電鉄オリジナル音声、または利用許諾のある音声に差し替えてください。ファイル名を保てばコード変更なしで差し替えできます。

アナウンス文は `data/announcements.js` に分離しています。`script.js` にはアナウンス文章を直接埋め込んでいません。
